
## Default Variables

### spctl_disable_master

Disable/Enable master system policy

#### Default value

```YAML
spctl_disable_master: true
```

### spctl_gatekeeper_policy

Update gatekeeper policy

#### Default value

```YAML
spctl_gatekeeper_policy: false
```
